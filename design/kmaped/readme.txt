
enchant向けマップエディタ kmaped です。

- 編集方法
　ボタンのバルーンの説明に従えば何とかなると思います。
　表示中の一番上のレイヤが編集対象となります。

- 保存方法
　ブラウザの localStrage に保存します。
　ファイルのダウンロードでローカルに持ってくる事もできます。
　フォーマットはJSON形式です。

- 読込方法
　付属の kmaped.enchant.js プラグインを使って読み込みます。
　使用例は test.html を参照してください。
　localStrage からでもマップを読み込めるのでエディタで保存すればすぐゲーム画面で確認できます。

- 捕捉
　レイヤに名前を付けられるようになっていますがこれは enchant 側で複数の Map に分割するような場合必要になります。
　通常の使い方では変更不要です。

- 実行環境
　kmaped.html と kmaped/** があればエディタは動作します。
　開発対象ゲームの index.html が有るディレクトリに配置して同一サーバからアクセスしてください。
　でないとlocalStrageを共有できません。
